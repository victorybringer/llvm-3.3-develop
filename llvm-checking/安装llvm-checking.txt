----------------安装过程--------------
1、上传 haskell-platform，llvm-checking

2、安装haskell-platform，运行ghci，如果提示找不到libgmp.so，apt-get install libgmp-dev

3、上传14.04  apt-get 清华源

4、apt-get install llvm-3.3, clang-3.3, g++

5、运行cabal update 后，上传 config 到 root/.cabal（其他用户根据首次运行cabal update后显示路径查看，一般都在该用户目录下的.cabal文件夹中），再运行 cabal update

6、安装 llvm-base， data， analysis， tool （0.3.0）

7、进入foreign-inference/目录（在lib下），安装archive，hbgl, svm, iiglue(这个依赖foreign-inference，安装foreign-inference后可以装)

8、安装HaXml-1.23.3

9、安装foreign-inference 0.3.5，如果有依赖问题，回头再安装一遍llvm-analysis目录的base，data，analysis，tool。还有能提示找不到libtinfo，可以 apt-get install libtinfo-dev

10、加载src中的Main.hs，如果有问题，encoding的问题的话，就安装encoding-0.8.5，还有CVS问题的话，就安装cassava-0.5.2.0
如果中途出现一些什么需要安装的，就正常安装，如c2hs，libtinfo-dev。

注意必须使用unordered-containers-0.2.8.0，等所有都装好了，才可以换成unordered-containers-0.2.9.0，然后所有的库重新装一下。unordered-containers-0.2.8.0也可以使用，只是在Defcet.hs里添加：
instance (Ord a) => Ord (HashSet a) where
    compare h1 h2 = compare (HS.toList h1) (HS.toList h2)
